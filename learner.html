Accuracy

 Accuracy (average:Optional[str]='micro', num_classes:Optional[int]=None,
           k:int=1, device:Optional[torch.device]=None)

This class is a wrapper for torchval.metrics.MulticlassAccuracy. It receives as input the prediction of the model and the target and: - finds the index of the maximum value in the prediction - aka the class - squizes the target to remove the last dimension
